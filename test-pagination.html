Accessibility Features

Keyboard Navigation

  • Use Tab to focus the pagination component
  • Use Tab to navigate between pagination buttons
  • Press Enter or Space to activate buttons
  • All buttons have proper ARIA labels for screen readers

Screen Reader Support

  • Navigation role with proper aria-label
  • Current page marked with aria-current="page"
  • Descriptive aria-labels for all buttons
  • Support for aria-describedby for additional context

Code Examples

Basic Usage

<ds-pagination current-page="3" total-pages="10" ></ds-pagination>

With All Options

<ds-pagination variant="minimal" size="large" current-page="5" total-pages="20" show-first-last max-visible-pages="5" aria-label="Custom pagination" ></ds-pagination>

With Total Items

<ds-pagination current-page="3" total-items="100" items-per-page="10" ></ds-pagination>

Event Handling

<ds-pagination current-page="3" total-pages="10" @ds-pagination-navigate="handleNavigate" ></ds-pagination> <script> function handleNavigate(event) { console.log('Navigate to page:', event.detail.data.page); } </script>
