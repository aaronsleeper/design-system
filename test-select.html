Code Examples

Basic Usage

<ds-select placeholder="Choose an option" .options="${options}" ></ds-select>

Multiple Selection

<ds-select type="multiple" placeholder="Choose multiple options" .options="${options}" ></ds-select>

Searchable Selection

<ds-select type="searchable" placeholder="Search and select" .options="${options}" ></ds-select>

With Validation

<ds-select placeholder="Required field" required validation-state="error" error-message="Please select an option" .options="${options}" ></ds-select>
